# Free vs Pro boundaries

Clay Performance Lab is currently in a closed beta. This document defines the future product boundary without making pricing, checkout or upgrade prompts visible to beta users.

## Free

Free should help shooters start logging and understand basic progress:

- Manual result logging.
- Basic result history.
- Basic training logs.
- Basic performance summary.
- Current-season Leirdue import.
- Basic result vs winner where a winner score exists.
- Basic offline logging for range use.
- Basic shooting ground cleanup for personal names and aliases.
- Compliance access to a user's own raw personal data.

## Pro

Pro should unlock improvement, interpretation, coaching value and packaged exports:

- Full historical Leirdue import.
- Advanced Performance trends.
- Ground trends.
- Miss-pattern analysis.
- Target-type analysis.
- Equipment comparison.
- Shared and organizer training score sheets.
- Coach Report generation and export.
- Coach-ready product exports.
- Advanced CSV and PDF exports.
- All AI-powered insights.
- AI training recommendations.
- AI app copilot.
- AI miss-pattern interpretation.

## Beta rule

The current closed beta must not show public pricing, checkout, commercial paywalls, upgrade buttons or Pro upsells by default. Pro gates may be prepared internally, and AI or other paid-cost features must have server-side entitlement guard paths before a public free launch.

## Data export distinction

Raw compliance access to a user's own personal data is not a paid product export. Coach-ready exports, PDFs, advanced CSV files and packaged analysis exports are Pro product features. This distinction must not block account compliance or user data safety support paths.
