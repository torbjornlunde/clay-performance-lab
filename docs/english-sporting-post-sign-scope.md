# English Sporting post-sign support

This first step formalizes English Sporting as a post-based sporting discipline using the existing per-stand target editor and post-sign photo analysis flow.

Included:
- English Sporting and the Norwegian alias `engelsk sporting` route to the post-based target editor.
- UI terminology uses `Stand` rather than `Post`.
- Compak Sporting and Sporttrap remain outside this flow until their own discipline-specific models are added.
- Regression checks protect the discipline routing and terminology.
